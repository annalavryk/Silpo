# 🛒 SILPO Ordering System

**SILPO Ordering System** — це консольна програма на мові C++, яка імітує роботу автоматизованого робочого місця касира або системи самообслуговування. Проєкт демонструє роботу з об'єктно-орієнтованою логікою, керування станами замовлення та збереження даних у зовнішні файли.

---

## 📖 Опис проєкту

Програма дозволяє користувачеві повністю пройти шлях від вибору продуктів до отримання чека. Вона враховує дату для нарахування знижок, підтримує промокоди та імітує логістичні процеси (пакування, доставку) у реальному часі.

### 🚀 Основний функціонал

* 🛒 **Гнучкий кошик:** Додавання товарів за ID, видалення за назвою та редагування кількості.
* 📅 **Товар дня:** Автоматичне визначення акційного товару (знижка **-10%**) за алгоритмом на основі введеної дати.
* 💾 **Персистентність:** Збереження кошика та статусу замовлення у файл `silpo.txt` та можливість їх завантаження.
* 💳 **Система оплати:** Вибір між карткою, готівкою або онлайн-оплатою.
* 🔄 **Логістичний трекер:** Анімована зміна статусів замовлення від "Оплачено" до "Доставлено".

## 🎟 Система промокодів

У програмі реалізовано перевірку секретних кодів для отримання додаткових знижок:

| Промокод | Ефект |
| :--- | :--- |
| **SALE10** | `-10%` на загальну суму |
| **SALE20** | `-20%` на загальну суму |

🛠 Технології та інструменти
Мова: C++ (стандарт C++17)

IDE: Visual Studio 2022 / GCC

Ключові бібліотеки:

<vector> — керування динамічним списком продуктів.

<fstream> — файлова система для збереження замовлень.

<thread> & <chrono> — для створення пауз у консолі (імітація процесів).
